Терминал
----------------------------------------------------------------------------------------------------
Для подключения к терминалу используем PUTTY/KiTTY
Включить опцию "Implicit CR", Отключить опцию "Implicit LF"
Сервер telnet localhost:19021 (JLink GDB Server)

Пример настройки цветового выделения текста: https://habrahabr.ru/post/119436/

Список команд:
task - получение списка задач
Предпоследний столбец, свободный объем памяти для задачи
memuse - возвращает количество свободной памяти ОЗУ
