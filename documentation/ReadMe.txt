Синхронизация проектов в Windows и Linux
-------------------------------------------------------------------------------
1. Запустить виртуальную машину VirtualBox
2. После запуска кликнуть по надписи "user"
3. Запустить терминал
3.1. Примонтировать директорию с проектом выполнив команду
     "sudo mount -t vboxsf 505.013.1000_KSU-02 /mnt/hgfs/505.013.1000_KSU-02" Пароль: q12345
3.2. Перейти в директорию выполнив команду
     "cd /mnt/hgfs/505.013.1000_KSU-02/AM335x_Main/tools/"
3.3. Выполнить синхринизацию проекта из Windows в Linux выполнив команду
     "sudo sh ./run_sync_linux.sh"

Заливка проекта в модуль
-------------------------------------------------------------------------------
1. Запустить ssh клиент
2. Подключиться к плате login: "root" password: "root"
3. Завершить процесс выполнив команду "killall ksu_gui"
